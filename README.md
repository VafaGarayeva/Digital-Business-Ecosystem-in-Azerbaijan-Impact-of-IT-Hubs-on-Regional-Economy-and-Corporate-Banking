# Digital-Business-Ecosystem-in-Azerbaijan-Impact-of-IT-Hubs-on-Regional-Economy-and-Corporate-Banking
This project analyzes the digital transformation landscape in Azerbaijan, focusing on how regional IT Hubs influence non-oil economic growth and Corporate Banking services.
### 🛠 Tech Stack

**Data Analysis:** Python (Pandas, NumPy)

**Business Intelligence:** Power BI (Data Modeling & DAX)

**Database:** SQL (Oracle)
1. Mapping regional growth through IT infrastructure.
2. Evaluating digital banking efficiency for corporate clients.
   **Python:** Used for data cleaning and pre-processing.
   **Power BI:** Created interactive dashboards to track KPIs.
